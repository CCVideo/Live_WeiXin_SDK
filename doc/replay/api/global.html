showEm(str) → {String}

聊天表情和链接处理方法。 需要发送带链接的a标签时，格式如下：[uri_http://www.xxx.com] （注意：前后都需要加空格）。 在收发消息的时候，用showEm('消息内容')处理一下消息内容即可，原理就是匹配[em2_*]，让它变为对应的表情图片，匹配[uri _ *]，让它变为可点击的a链接。
Parameters:
Name Type Description
str String 聊天内容
Source:
  • replay/util.js, line 52
Returns:
返回处理后的聊天信息，使用a、img标签处理表情和链接，使用rich-text进行渲染。
Type
String
Example
cc.replay.showEm('大家好我是CC视频[em2_01]');
